# 🎬 Movie Recommendation System using ANN

## 📌 Project Overview
This project is a Movie Recommendation System built using Artificial Neural Networks (ANN). It recommends movies to users based on their preferences and historical ratings.

The model is trained on the MovieLens dataset and learns latent user-movie relationships to generate personalized recommendations.

---

## 🚀 Features

- Movie recommendation using ANN
- User-Movie interaction learning
- Data preprocessing pipeline
- Model training and evaluation
- Top-N movie recommendations
- Scalable architecture
